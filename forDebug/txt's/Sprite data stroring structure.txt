Sprite struct:
- sp[n]
- id
- type
- state
- vec3 pos

Sprite tulajdonságok:
- sp[n]: Hanyadik sprite
- sp[n].id: Milyen textúra tartozik hozzá (fordítóhoz kell)
- sp[n].type:
	- Ha item, felvehető, stat módosító (kulcs, heal, ammo)
	- Ha enemy, A* pathfinding
	- Ha fény, megvilágítás
	- ha object, hozzá illő metódus (Pl: chest -> kinyitható, stat módosító)
- sp[n].state: On/Off - Rajzoljuk vagy sem
- sp[n].pos.x,y,z: Vertikális, horizontális és mélységi pozíció

Sprite rajzolás:
(Fragment shader)
- X1, Y1,

- X1: sp[n].pos.x - playerPosition.x
- Y1: sp[n].pos.y - playerPosition.y
- Z: sp[n].pos.z

pA = playerAngle(rad)

//Rotation matrix
cos = cos(pa)
sin = sin(pa)
X1 = Y1*cos+X1*sin
Y1 = X1*cos-Y1*sin

//Screen coordinates
clientSize.x,y
random const.

X1 = (X1 * const / Y1) + (sp[n].pos.z / 2)
